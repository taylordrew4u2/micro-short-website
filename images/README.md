# Film Stills

Place the following image files in this directory to populate the page backgrounds:

| File | Page |
|------|------|
| `still-home.jpg`    | Home / Landing page |
| `still-cast.jpg`    | Cast page |
| `still-crew.jpg`    | Crew page |
| `still-awards.jpg`  | Awards page |
| `still-contact.jpg` | Contact page |

Images should be high-resolution (1920 × 1080 px or larger) and in JPEG or WebP format.
If an image is absent, the corresponding page falls back to a cinematic gradient defined in `css/style.css`.
